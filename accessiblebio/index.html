My Biography

Hello, I'm Dylan Rodrigues. I am passionate about cybersecurity and accessibility, with the goal of building secure digital experiences that are inclusive for everyone.

In terms of accessibility, I hope to improve web standards and create security standards like authentication and data protection systems that are accessible to all.

My Interests in Accessibility

Accessibility is vital to ensure that everyone can use the web. Some key aspects that interest me include:

  • Using semantic HTML to support screen readers
  • Providing meaningful alternative text for images
  • Ensuring sufficient color contrast
  • Improving keyboard navigation
